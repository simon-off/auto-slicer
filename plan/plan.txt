--- LITEN PLAN BARA ---

Om sample-styrkan (volymen) går under THRESHOLD i TAIL sekunder: Klipp ljudfilen efter TAIL och spara.
När sample-styrkan (volymen) går över THRESHOLD igen: Skapa ny ljudfil.


### OPTIONS ###

THRESHOLD (-100 - 0.db)     Numeric input
TAIL: (0.1 - 20s)           Numeric input
FADE IN: (0 - 1s)           Numeric input
FADE OUT: (0 - 10s)	        Numeric input
CHANNEL-DATA: L, R          Radio


### BUTTONS ###
IMPORT
EXPORT
